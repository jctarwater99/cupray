Installing React native for Developing
	> node -v 		// (if version is less than 12.14.1 or dne, update it, good luck)
	> npm install -g expo-cli
	> // yarn add expo??
			// Also install expo client app on mobile phone

	Creating a new project is done by running
	> expo init <nameOfProject>
	But you can probably just pull from the repo?

Installing Sqlite for the database
	> expo install expo-sqlite

Installing radio buttons to simplify user error
	> npm install radio-buttons-react-native --save

Installing expo notification handling
	> expo install expo-notifications

Installing React native navigation for multiple screens?
	> npm install @react-navigation/native
	> npm install @react-navigation/stack
